# 🚀 Быстрый старт для новичков в Django

Этот файл поможет вам быстро запустить проект и понять, как он работает.

## ⚡ Быстрый запуск (5 минут)

### 1. Откройте терминал в папке проекта
```bash
cd pythonProject
```

### 2. Установите зависимости
```bash
pip install poetry
poetry install
```

### 3. Создайте базу данных
```bash
poetry shell
python manage.py makemigrations
python manage.py migrate
```

### 4. Создайте тестовые данные
```bash
python run_demo.py
```

### 5. Запустите сервер
```bash
python manage.py runserver
```

### 6. Откройте браузер
Перейдите по адресу: **http://127.0.0.1:8000/**

🎉 **Готово! Ваш сайт работает!**

## 🔍 Что посмотреть

## 🧪 Эксперименты для новичков

### 1. Измените количество дней
Откройте файл `booking/views.py` и найдите строку:
```python
all_dates = [today + timedelta(days=x) for x in range(30)]
```
Измените `30` на `7` и перезапустите сервер. Что изменилось?

### 2. Добавьте новый стол
В админке Django или через Django shell:
```python
python manage.py shell
from booking.models import Table
Table.objects.create(number=10, capacity=12, is_active=True)
```

### 3. Измените цвета
Откройте `templates/free_dates.html` и найдите CSS:
```css
.date-item.free {
    background: #e8f5e8;  /* Зеленый цвет */
}
```
Измените на любой другой цвет, например `#ffeb3b` (желтый).

## ❓ Частые вопросы

### Q: Что означает ошибка "No module named 'django'"?
**A:** Django не установлен. Выполните `poetry install` или `pip install django`.

### Q: Страница не загружается, ошибка 404?
**A:** Проверьте, что сервер запущен командой `python manage.py runserver`.

### Q: База данных не создается?
**A:** Убедитесь, что выполнили `python manage.py migrate`.

### Q: Как добавить новый функционал?
**A:** Изучите структуру проекта и добавьте новые модели, представления и URL.
