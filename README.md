# 🍽️ Система бронирования столиков

Веб-приложение для бронирования столиков в ресторане, построенное на Django с использованием Django REST Framework.

## 🆕 Вы новичок в Django?

Если вы впервые работаете с Django, начните с изучения файла **[`DJANGO_FOR_BEGINNERS.md`](DJANGO_FOR_BEGINNERS.md)** - там подробно объяснены все основные концепции на примерах этого проекта!

## ✨ Возможности

- 📅 Просмотр свободных дат для всех столов
- 🪑 Информация о вместимости каждого стола
- 🔄 API для получения данных о доступности
- 📱 Адаптивный дизайн для всех устройств
- 🎨 Современный и красивый интерфейс

## 🚀 Установка и запуск

### 1. Установка зависимостей

```bash
# Установка Poetry (если не установлен)
pip install poetry

# Установка зависимостей проекта
poetry install
```

### 2. Настройка базы данных

```bash
# Активация виртуального окружения
poetry shell

# Создание миграций
python manage.py makemigrations

# Применение миграций
python manage.py migrate
```

### 3. Создание тестовых данных

```bash
# Создание суперпользователя (опционально)
python manage.py createsuperuser

# Создание тестовых данных
python manage.py shell < create_test_data.py
```

### 4. Запуск сервера

```bash
python manage.py runserver
```

После запуска откройте браузер и перейдите по адресу: http://127.0.0.1:8000/

## 🔌 API Endpoints

### Получить свободные даты для конкретного стола
```
GET /api/tables/{id}/free-dates/
```

### Получить свободные даты для всех столов
```
GET /api/tables/all-free-dates/
```

### Получить список всех столов
```
GET /api/tables/
```

## 🗄️ Модели данных

### Table (Стол)
- `number` - номер стола
- `capacity` - вместимость (количество мест)
- `image` - изображение стола (опционально)
- `is_active` - активность стола

### Booking (Бронирование)
- `booking_date` - дата бронирования
- `table` - связанный стол
- `user` - пользователь, сделавший бронь
- `created_date` - дата создания брони

## 🎨 Особенности интерфейса

- **Адаптивный дизайн** - работает на всех устройствах
- **Цветовая индикация** - зеленый для свободных дат, красный для забронированных
- **Статистика** - общее количество столов и свободных дат
- **Интерактивность** - hover эффекты и анимации
- **Обновление данных** - кнопка для обновления информации

## 🧪 Тестирование

Для тестирования API можно использовать:

1. **Браузер** - просто откройте URL в браузере
2. **Postman** - для тестирования API endpoints
3. **curl** - для командной строки

Пример curl запроса:
```bash
curl http://127.0.0.1:8000/api/tables/all-free-dates/
```

## 🔧 Настройка

Основные настройки находятся в файле `config/settings.py`:

- `DEBUG` - режим отладки
- `DATABASES` - настройки базы данных
- `MEDIA_URL` - URL для медиа файлов
- `STATIC_URL` - URL для статических файлов
